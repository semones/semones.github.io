【效率备忘】Microsoft VS Code 配置推荐

Semones published on 2025-04-13 included in 效率备忘

🛠️ 全局设置

1. 程序设置

// 程序设置
"update.enableWindowsBackgroundUpdates": false,	// 禁止后台下载更新
"update.mode": "none",							// 禁止任何更新行为
"extensions.autoCheckUpdates": false,           // 禁止插件检查更新
"extensions.autoUpdate": false,                 // 禁止插件自动更新
"extensions.ignoreRecommendations": true,		// 关闭插件安装建议
"files.autoGuessEncoding": true,				// 打开文件时猜测字符集编码
"explorer.compactFolders": false,				// 关闭文件夹紧凑显示模式
"chat.agentsControl.enabled": "hidden",			// 关闭 Copilot 代理控制功能

2. 工作区设置

// 工作区设置
"workbench.startupEditor": "none",				// 关闭启动页面
"workbench.editor.empty.hint": "hidden",		// 编辑器为空时，不提示打开文件
"workbench.list.smoothScrolling": true,			// 开启界面平滑滚动效果
"workbench.colorTheme": "Dracula Theme Soft",	// 用 Dracula Theme Official 插件设置主题外观
"workbench.iconTheme": "material-icon-theme",	// 用 Material Icon Theme 插件设置图标样式

3. 编辑器设置

// 编辑器设置
"editor.fontSize": 18,							// 字体大小
"editor.mouseWheelZoom": true,					// 允许使用 ctrl + 滚轮缩放字体大小
"editor.fontFamily": "Consolas, '微软雅黑'",	 // 设置字体
"editor.smoothScrolling": true,					// 开启编辑器区域平滑滚动效果
"editor.cursorSmoothCaretAnimation": "on",		// 开启光标平滑移动效果
"editor.cursorBlinking": "smooth",				// 开启光标淡入淡出效果
"editor.fontLigatures": false,					// 关闭字体连字效果

4. 终端设置

// 终端设置
"terminal.integrated.smoothScrolling": true,					// 开启终端平滑滚动效果
"terminal.integrated.defaultProfile.windows": "PowerShell",		// 设置终端默认启动 PowerShell
"terminal.integrated.profiles.windows": {						// 终端详细配置
    "PowerShell": {
        "path": "C:\\Windows\\System32\\WindowsPowerShell\\v1.0\\powershell.exe"
    },
    "Command Prompt": {
        "path": "C:\\Windows\\System32\\cmd.exe",
        "args": [],
        "icon": "terminal-cmd"
    },
    "Git Bash": {
        "source": "Git Bash",
        "icon": "terminal-git-bash"
    }
},

【算法笔记】采购订单

Semones published on 2024-03-22 included in 算法笔记

⚙️ 基本设定

1. 规则描述

  • 采购系统中，采购申请 (PR) 要批准后才能生成采购订单 (PO)，0 表示审批通过，1 表示审批拒绝，2 表示待审批。
  • 单价高于 100 元的商品，每次申请单独生成一条 PO；单价相同且小于等于 100 元的相同商品合并到一条 PO。
  • 商品单价小于 100 元且数量大于等于 100 的商品单价打九折。
  • 按商品 ID 升序排序，ID 相同则按数量降序排列。

2. 输入描述

  • 第一行输入 PR 的数量 n。
  • 接下来 n 行，每行输入四个整数表示商品 ID、数量、单价、PR 状态。

3. 输出描述

  • 商品 ID、数量、单价。

【编程随笔】对于 C 语言指针的理解与想法

Semones published on 2024-01-01 included in 编程随笔

理解一门语言要从底层入手。

1. 地址

C 语言的编译器会把.c文件编译为机器码存入一个.exe文件，机器码可以对应汇编语言，是直接与 CPU 对话的语言。各类操作系统比如 Windows、Linux 会提供 API 支持 C 程序的执行，可以看作一个巨大的“运行环境”。对比 Java 的编译器会把.java文件编译为字节码存入一个.class文件，其实思路是相似的，字节码是与 JVM 对话的语言，JVM 就是一个虚拟机，模拟了操作系统和 CPU，它需要把字节码翻译成各种机器码与不同架构的 CPU 对话。JVM 是运行在系统上的，而 Java 程序是运行在 JVM 上的。
