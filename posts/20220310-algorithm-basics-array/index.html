1. 二分法

(1) 基本要求
  1. 前提条件：返回无重复元素的有序数组中的指定的值的下标。
  2. 工作原理：每次与中间元素比较，根据比较结果决定继续搜索左半部分还是右半部分。
(2) 代码实现

基于升序数组

  1. 迭代
public class BinarySearchIteration {
	//迭代
	public static int binarySearchIteration(int[] nums, int target) {
		if (target < nums[0] || target > nums[nums.length - 1]) {
            return -1;
        }

		int left = 0;
		int right = nums.length - 1;

		while (left <= right) {
			int mid = left + (right - left) / 2;

			if (target == nums[mid]) {
				return mid;
			} else if (target < nums[mid]) {
				right = mid - 1;
			} else {
				left = mid + 1;
			}
		}

		return -1;
	}
}
  1. 递归
public class BinarySearchRecursive {
	//递归
	public static int binarySearchRecursive(int[] nums, int target, int left, int right) {
		if (left > right) {
			return -1;
		}
		if (target < nums[0] || target > nums[nums.length - 1]) {
			return -1;
		}

		int mid = left + (right - left) / 2;

		if (nums[mid] == target) {
            return mid;
        } else if (target < nums[mid]) {
            return binarySearchRecursive(nums, target, left, mid - 1);
        } else {
            return binarySearchRecursive(nums, target, mid + 1, right);
        }
	}
}
(3) 细节分析
  1. 如果查询的数不在数组内，会先产生left = right = mid，接着下一步会造成left > right从而完成判断，所以while (left <= right)是<=。
  2. mid = left + (right - left) / 2是为了防止溢出。如果是mid = (left + right) / 2，两数的和可能超过整型上限。

2. 移除元素

(1) 基本要求
  1. 前提条件：移除指定元素，不使用额外数组空间，可改变顺序。函数返回新的长度。
  2. 工作原理：两个指针 (索引) 配合。
(2) 代码实现
  1. 快慢指针
public class RemoveElementFastSlowPointer {
	//快慢指针
	public int removeElementFastSlowPointer(int[] nums, int val) {
		int slow = 0;

		for (int fast = 0; fast < nums.length; fast++) {
			if (nums[fast] != val) {
				nums[slow] = nums[fast];
				slow++;
			}
		}

		return slow;
	}
}
  1. 左右指针
public class RemoveElementLeftRightPointer {
	//左右指针
	public int removeElementLeftRightPointer(int[] nums, int val) {
		int left = 0;
        int right = nums.length - 1;

        while(left <= right) {
            if (nums[left] == val) {
                nums[left] = nums[right];
                right--;
            } else {
                left++;
            }
        }

        return left;
	}
}
(3) 细节分析
  1. 第一个指针找val，第二个指针找非val，把第二个指针覆盖第一个。两种方法是从左向右和从右向左的区别。
  2. 左右指针中，语句else {left++};确保覆盖左指针位的值等于val，即使右指针赋给左指针val，左指针也不会自增。

3. 有序数组的平方

(1) 基本要求
  1. 前提条件：非递减顺序。
  2. 工作原理：非递减顺序的数组两端绝对值大，两个指针从两端向中心移动。
(2) 代码实现
public class SortedSquares {
	//双指针
	public int[] sortedSquares(int[] nums) {
		int[] result = new int[nums.length];
		int index = nums.length - 1;
		int left = 0;
		int right = nums.length - 1;

		while (left <= right) {
			if (nums[left] * nums[left] > nums[right] * nums[right]) {
				result[index--] = nums[left] * nums[left];
				left++;
			} else {
				result[index--] = nums[right] * nums[right];
				right--;
			}
		}

		return result;
	}
}
(3) 细节分析
  1. index--返回的值是index，然后再自减。

4. 长度最小的连续子数组

(1) 基本要求
  1. 前提条件：和大于等于给定的数，返回的是长度。
  2. 工作原理：滑动窗口。
(2) 代码实现
public class MinSubArrayLen {
	//滑动窗口
	public int minSubArrayLen(int[] nums, int s) {
		int left = 0;
		int sum = 0;
		int result = Integer.MAX_VALUE;

		for (int right = 0; right < nums.length; right++) {
			sum += nums[right];
			
			while (sum >= s) {
				result = Math.min(result, right - left + 1);
				sum -= nums[left++];
			}
		}

		return result == Integer.MAX_VALUE ? 0 : result;
	}
}
(3) 细节分析
  1. sum大于等于s时，可能减去开始的元素仍大于等于s。
  2. 跳出whilw循环前sum一定会小于s，恰好right++启动右指针计算后续元素。

5. 螺旋矩阵

(1) 基本要求
  1. 前提条件：生成 1 到 n^2 按顺时针顺序螺旋排列的正方形矩阵。
  2. 工作原理：循环不变量。按圈写入数组，左闭右开分为四条边。两个指针，i指向每一行，沿列方向移动；j指向每一列，沿行方向移动。
(2) 代码实现
public class GenerateMatrix{

	public int[][] generateMatrix(int n) {
		int[][] matrix = new int[n][n];
		int startR = 0, startC = 0;  //Row & Column
		int offset = 1;
		int count = 1;
		int loop = 1;
		int i, j;

		while (loop <= n / 2) {
			for (j = startC; j < n - offset; j++) {
                matrix[startR][j] = count++;
            }
            for (i = startR; i < n - offset; i++) {
            	matrix[i][j] = count++;
            }
            for (; j > startC; j--) {
            	matrix[i][j] = count++;
            }
            for (; i > startR; i--) {
            	matrix[i][j] = count++;
            }

            startR++;
            startC++;
            offset++;
            loop++;
		}
		if (n % 2 == 1) {
			matrix[startR][startC] = count;
		}

		return matrix;
	}
}
(3) 细节分析
  1. 每圈的起点沿左上对角线向中心靠拢，startC与startR指向起点，自增以缩圈。
  2. 起点在到达中心点前停止，所以有loop <= n / 2。
  3. i与j分别停止等待对方走完列或行，接力转圈。

6. 区间和

(1) 基本要求
  1. 前提条件：先输入数组长度，再依次输入数组元素，接着输入要求的区间，求和。
  2. 工作原理：前缀和。建立新数组记录原数组 0 到 i 的累加和。
(2) 代码实现
import java.util.Scanner;

public class IntervalSum{
	//前缀和
	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);

		int n = scanner.nextInt();
		int[] nums = new int[n];
		int[] prefixSumArr = new int[n];
		int prefixSum = 0;

		for (int i = 0; i < n; i++) {
			nums[i] = scanner.nextInt();
			prefixSum += nums[i];
			prefixSumArr[i] = prefixSum;
		}

		while (scanner.hasNextInt()) {
			int a = scanner.nextInt();
            int b = scanner.nextInt();
            int sum;

            if (a == 0) {
                sum = prefixSumArr[b];
            } else {
                sum = prefixSumArr[b] - prefixSumArr[a - 1];
            }
            
            System.out.println(sum);
		}

		scanner.close();
	}
}
(3) 细节分析
  1. 在计算机领域，prefix 指开头到某个位置的连续部分。
  2. sum = prefixSumArr[b] - prefixSumArr[a - 1];是一个简单的数学问题，将复杂度降低至 O(1)。
  3. scanner.hasNextInt()检查下一个输入是否为整数，但不会实际读取。可以被后面的指令读取。
  4. scanner.close();关闭Scanner对象释放资源。

7. 土地分配

(1) 基本要求
  1. 一个矩形区域有 n * m 个区块，把矩形横向或纵向分割成两个区域，计算最小权值差。
  2. 前缀和，按行列整体计算。
(2) 代码实现
import java.util.Scanner;

public class LandAllocation {
	//前缀和
	public static void main() {
		Scanner scanner = new Scanner(System.in);
		int n = scanner.nextInt();
		int m = scanner.nextInt();
		int sum = 0;
		int[][] grid = new int[n][m];

		for (int i = 0; i < n; i++) {
			for (int j = 0; j < m; j++) {
				grid[i][j] = scanner.nextInt();
				sum += grid[i][j];
			}
		}

		int[] horizontal = new int[n];
		for (int i = 0; i < n; i++) {
			for (int j = 0; j < m; j++) {
				horizontal[i] += grid[i][j];
			}
		}

		int[] vertical = new int[m];
		for (int j = 0; j < m; j++) {
			for (int i = 0; i < n; i++) {
				vertical[j] += grid[i][j];
			}
		}

		int result = Integer.MAX_VALUE;

		int horizontalCut = 0;
		for (int i = 0; i < n - 1; i++) {
			horizontalCut += horizontal[i];
			result = Math.min(result, Math.abs((sum - horizontalCut) - horizontalCut));
		}

		int verticalCut = 0;
		for (int j = 0; j < m - 1; j++) {
			verticalCut += vertical[j];
			result = Math.min(result, Math.abs((sum - verticalCut) - verticalCut));
		}

		System.out.println(result);
		scanner.close();
	}
}
(3) 细节分析
  1. 两区域至少有一个区块，所以计算子区域时的i与j要小于n - 1。
  2. 提前求总和，用差来算另一个区域，减少计算量。
