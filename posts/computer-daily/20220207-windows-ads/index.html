解决“此文件来自其他计算机，可能被阻止以帮助保护该计算机”的问题

Semones included in 电脑日常
 2022-02-07  614 words   2 minutes 
Contents
    • 1. 解锁方法
    • 2. 杜绝方法
    • 3. 原理简述

有时候从网上下载的文件属性会有一条“此文件来自其他计算机，可能被阻止以帮助保护该计算机”的安全锁定信息，这对强迫症很不友好，本文会介绍一种简单的方法来解锁，并且会提供一种方案杜绝这条信息的出现，在最后会简单解释相关原理。

1. 解锁方法

  1. 先在微软官网下载流工具包：https://learn.microsoft.com/zh-cn/sysinternals/downloads/streams
  2. 解压后把 streams.exe 放到与待解锁文件的相同文件夹中，在此文件夹打开 cmd 窗口 (地址栏输入 cmd 后回车)
  3. streams.exe -s		#批量查看包括子文件夹中哪些文件被锁定
streams.exe -s -d	#批量解锁包括子文件夹中的文件

2. 杜绝方法

  1. Win + R 打开运行，输入 gpedit.msc 后回车打开本地组策略编辑器；
  2. 依次选择 用户配置 → 管理模板 → Windows 组件 → 附件管理器；
  3. 双击打开“文件附件中不保留区域信息”这一项，选择“已启用”后确认，这样网上下载的文件就不会出现这条信息。

3. 原理简述

流 (Alternate Data Streams) 是 NTFS 文件系统的一个高级特性，允许一个文件包含多个独立的数据流。每个 NTFS 文件至少有一个主数据流，可以有多个命名数据流。例如：

文件：document.docx
├── 主数据流 - 文档内容
├── Zone.Identifier - 安全标记
├── SummaryInformation - 属性摘要
└── 其他自定义流

其中 Zone.Identifier 是 Windows 用于标记文件来源安全区域的元数据文件。用户从未知站点下载文件，系统会自动添加 Zone.Identifier 流并设置 ZoneId = 3 (0-本地，1-内部网，2-可信站点，3-互联网，4-受限站点)，属性里便会出现这条安全警告，本意是为提高用户安全意识。
