算法基础 2. 双指针移除数组元素

Semones included in 算法基础
 2022-03-23  404 words   One minute 
Contents
    • 1. 基本要求
    • 2. 代码实现

使用两个指针配合，在一个 for 循环下完成两个 for 循环的工作。

1. 基本要求

  1. 移除指定元素，不使用额外数组空间，可改变顺序。函数返回新的长度。
  2. 数组的元素在内存地址中是连续的，不能单独删除数组中的某个元素，只能覆盖。

2. 代码实现

  1. 快慢指针
public class RemoveElementsFastSlowPointer {
	// 快慢指针
	public int getLength(int[] nums, int val) {
		int slow = 0;

		// 快指针遍历数组
		for (int fast = 0; fast < nums.length; fast++) {
			// 快指针扫描到目标元素时不给慢指针赋值，慢指针停留等待快指针扫描非目标元素来完成覆盖
			if (nums[fast] != val) {
				nums[slow] = nums[fast];
				slow++;
			}
		}

		// fast 每扫描到一个非目标元素，slow 自增，恰好是新数组长度
		return slow;
	}
}
  1. 左右指针
public class RemoveElementsLeftRightPointer {
	// 左右指针
	public int getLength(int[] nums, int val) {
        // 两指针从两边向中间靠拢
		int left = 0;
        int right = nums.length - 1;

        // 左指针自增，扫描到目标值用右指针的值覆盖后再次比较，事实上每个值都靠 left 比较
        while (left <= right) { // left == right 时，比较最后一个元素
            if (nums[left] == val) {
                nums[left] = nums[right];
                right--;
            } else { // 确保左指针的值不等于目标值，即使右指针也恰好把目标值赋给了左指针，也不会自增
                left++;
            }
        }

        // 只有非目标元素才会使 left 自增
        return left;
	}
}
