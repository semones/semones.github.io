算法基础 5. 前缀和

Semones included in 算法基础
 2022-05-28  394 words   One minute 
Contents
    • 1. 基本要求
    • 2. 代码实现

前缀和 (Prefix Sum) 数组的第 i 个元素表示原数组从第 0 个元素到第 i 个元素的累加和。

1. 基本要求

  1. 计算数组在指定区间的和。
  2. 先输入数组长度，再依次输入数组元素，接着输入要求的区间。

2. 代码实现

/*
  1. 如果暴力求解，查询 m 次，每次查询范围都是 0 到 n - 1，时间复杂度为 O(n * m)，会非常大。
  2. 前缀和可以把时间复杂度降为 O(1)。
  3. Java 控制台中，空格或回车分隔数据。一次性输入大量数据后，按照代码顺序依次读取。
*/

import java.util.Scanner;

public class IntervalSum{
	// 前缀和
	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);

		int n = scanner.nextInt();
		int[] nums = new int[n];
		// prefix 指开头到某个位置的连续部分。
		int prefixSum = 0; // 前缀和指数组中前 i 个元素的总和
		int[] prefixSumArr = new int[n]; // 前缀和的数组

		// 构建原数组和前缀和数组
		for (int i = 0; i < n; i++) {
			nums[i] = scanner.nextInt();
			prefixSum += nums[i];
			prefixSumArr[i] = prefixSum;
		}

		// hasNextInt() 检查下一个输入是否为整数，但不会实际读取
		// 实现未知数量输入处理
		while (scanner.hasNextInt()) {
			int a = scanner.nextInt();
            int b = scanner.nextInt();
            int sum;

            if (a == 0) { // 等价于前缀和
                sum = prefixSumArr[b];
            } else {
            	// 用前缀和求区间和
                sum = prefixSumArr[b] - prefixSumArr[a - 1];
            }
            
            System.out.println(sum);
		}

		// 关闭 Scanner 对象释放资源。
		scanner.close();
	}
}
