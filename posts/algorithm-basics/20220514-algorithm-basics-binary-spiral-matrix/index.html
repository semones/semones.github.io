算法基础 4. 生成螺旋矩阵

Semones included in 算法基础
 2022-05-14  366 words   One minute 
Contents

利用指针和偏移量达到螺旋效果。

1. 基本要求

  1. 按顺时针顺序生成 1 到 n^2 螺旋排列的正方形矩阵。
  2. 输出二维数组：
for (int[] row : array) {
	System.out.println(Arrays.toString(row));
}

2. 代码实现

public class SpiralMatrix{

	public int[][] getMatrix(int n) {
		int[][] matrix = new int[n][n];
		int startR = 0, startC = 0; // 行与列的序号构成每圈的起始点
		int offset = 1; // 左闭右开偏移量
		int count = 1; // 待填写的数字
		int loop = 1; // 当前圈数
		int i, j; // i 指向每一行，纵向移动；j 指向每一列，横向移动

		// 构建行时，要用指向列的 j 扫描；构建列时，要用指向行的 i 扫描
		// i 与 j 分别停止等待对方走完列或行，接力转圈。
		while (loop <= n / 2) { // 起始点从左上角沿对角线向中心靠拢，到达中心点前停止
			for (j = startC; j < n - offset; j++) {
                matrix[startR][j] = count++;
            }
            for (i = startR; i < n - offset; i++) {
            	matrix[i][j] = count++;
            }
            for (; j > startC; j--) {
            	matrix[i][j] = count++;
            }
            for (; i > startR; i--) {
            	matrix[i][startC] = count++;
            }

            // 起始点自增以缩圈
            startR++;
            startC++;
            offset++;
            loop++;
		}

		// 跳出 wihle 循环时，startR 与 startC 一定比 n / 2 大 1
		// 如果矩阵边长时奇数，会留下一个中心点
		if (n % 2 == 1) {
			matrix[startR][startC] = count;
		}

		return matrix;
	}
}
